# Portfolio Generator CLI

## Description
Portfolio Generator CLI is a Node.js command-line application that prompts the user for professional information and generates a styled HTML portfolio page. The application collects details such as name, profession, location, bio, skills, projects, LinkedIn URL, and GitHub URL, then writes the completed portfolio to the file system.

## Technologies Used
- Node.js
- Inquirer 8.2.4
- JavaScript
- fs module
- HTML and CSS

## Installation
1. Clone the repository.
2. Run `npm install`.
3. Start the application with `node index.js`.

## Usage
The user answers the prompts in the command line. After the prompts are completed, the application generates a `portfolio.html` file in the project folder.
